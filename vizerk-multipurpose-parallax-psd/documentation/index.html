Features

  • 4 Fully Layered and Editable Photoshop (PSD) files.
  • Main Blog Page
  • Single Blog Page
  • 1170 Grid Based Design
  • Option of Parallax and without Parallax
  • Clean & Modern Design
  • Web Friendly Fonts

PSD Files

There is 4 PSD Files included in this Template.

  • Main Landing Page Design with Parallax (01_vizerk-Main.psd)
  • Main Landing Page Design without Parallax (02_vizerk-No-Parallax.psd)
  • Main Blog Post Page (03_blog-Main.psd)
  • Single Post Blog Page (04_blog_Post.psd)

Each PSD file is fully layered, named and properly grouped into different categories for easy recognization. You can easily edit the PSD file and make necessary adjustments.

Font Info

This PSD Template uses Google Web Fonts which is free to download and use. Make sure you have these fonts are installed in your system before editing the PSD files.
